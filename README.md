# open-data-ai-analytics
Мета проєкту

Аналіз доходів населення України на основі відкритих даних

У проєкті реалізовано:

- завантаження даних
- перевірку якості даних
- дослідницький аналіз
- побудову моделей
- візуалізацію результатів

## Джерело даних

## Питання / гіпотези для аналізу

1. Як змінювався наявний дохід населення України з 2005 по 2021 рік?

2. Які регіони мають найбільший дохід у останній доступний рік?

3. Чи можна за допомогою простої моделі машинного навчання спрогнозувати подальший тренд доходів?
