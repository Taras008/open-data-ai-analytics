# open-data-ai-analytics
Мета проєкту
Дослідження динаміки доходів населення по регіонах України та оцінка тренду за допомогою простої моделі.

У проєкті реалізовано:
завантаження даних (data_load)
перевірку якості даних (data_quality_analysis)
дослідницький аналіз і побудову простої моделі (data_research)
візуалізацію результатів (visualization)

## Джерело даних
https://data.gov.ua/

## Питання для аналізу
1. Як змінювався наявний дохід населення України з 2005 по 2021 рік?
2. Які регіони мають найбільші значення доходу в останній доступний рік?
3. Чи можна за допомогою простої лінійної регресії оцінити тренд та отримати прогноз на наступний рік?
